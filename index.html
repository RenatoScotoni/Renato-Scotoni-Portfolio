Renato Scotoni's Programming Portfolio

This is the portfolio of yours truly Renato Scotoni, collecting the best projects I have worked on throughout a semester of intense studying and work in the frigid lands of Canada.

And of course, what better way to show off these marks of my knowledge than with a top 10?

Top 10

#10: Learn Git Branching

"Learn Git Branching" was a challenging experience revolving around the software Git. It explored most intricacies of its usage through flowcharts, allowing me to understand how it functions without the addition of GitHub. Thanks to this website, I can now say that I learned the most common version control tool in the world, which is a necessity for a modern developer.
However, as it only taught me in the abstract how to use a piece of software and not a language - what I learned from it does not directly help me code - I cannot rank it as high as other modules.

#9: GitHub Learning Lab Tutorials

The GitHub Learning Lab showed me where I could put my Git knowledge to good use: Microsoft's hosting provider is perhaps the most crucial site for a developer to familiarize themselves with - after all, they're hosting this very portfolio! Their Learning Labs may not have been ideal, but they built upon Learn Git Branching's solid foundation to give me a solid rundown of the platform's kinks.
Ultimately, however, I cannot in good faith place Git-related modules on the same level as learning how to code.

#8: Khanacademy's Intro to JS

Javascript is one of the fundamental building blocks of the web, and Khanacademy's Intro to JS was all about it. During this course, I studied the basics of the language, achieving a good understanding of its functioning that allowed me to further my ability to develop websites.
However, this gave me a rather simple understanding of the basics of JS, and many parts of it - such as what I'm using in this very website - went unmentioned. As such, it ranks lowest among Khanacademy courses.

#7: Khanacademy's Advanced JS: Natural Simulations

In Khanacademy's Advanced JS: Natural Simulations, I learned how to utilize randomness, noise, trigonometry, particle systems, and physics to create programs which approximate reality. While the direct, real-world applications of this aren't obvious, the complexity of some of the projects definitely made me a better programmer.
In spite of the module was among the most engaging I have completed this semester, however, its unnecessary length made it, ultimately, an inefficient use of my time.

#6: Khanacademy's Intro to HTML/CSS

Khanacademy's intro to HTML and CSS taught me the two languages through which the Internet's less-interactive aspects are programmed. The CSS course was especially interesting, as it showed me how real, modern websites could be designed from a graphic point of view.
It ranks lower than it could, however, as the HTML section was rather boring and uncomplicated.

#5: Phaser Game Jam

During the Phaser Game Jam, I completed Codecademy's "Learn Game Development with Phaser.js" course and employed my knowledge of the game development-focused language to modify a pre-existing game, Codey Jump, in a team.
While group projects are usually exciting, having to speed through a course in a language I'm unlikely to use again to get to the interesting part was not, and the modification process itself was not as interesting as the process of making a new game from scratch would be have been.
However, these suboptimal conditions could be said to have prepared me for unexpected scenarios while at work.
Link to the github repository .

#4: Khanacademy's Advanced JS: Games and Visualizations

In Khanacademy's Advanced JS: Games and Visualizations, I learned how to create the titular games and visualizations in ProcessingJS through object-oriented programming, touching on transformations and 3D visualizations along the way. It was an enlightening course, deepening my knowledge of OOP and its applications and furthering my learning whilst not taking that long to complete. The 3D section was particularly eye-opening regarding the lower-level functioning of 3D engines.
For all these reasons, this is the most highly ranked Khanacademy course.

#3: Microbit Radio Project

During the course of this project, I and my team utilized the BBC's Microbit to create a rudimentry communication device. This device can send emojis, evaluate signal strength, and act as a simple accelerometer. Throughout this project I learned about the complexity of real-time communication and the limits of radio waves while improving my ability to work in a team.
However, the limited real-world applicability of what I learned means I cannot place this project in the top 2.
Link to the github repository .

#2: Mental Health Hackaton

During this team project, we developed a website compiling resources for those struggling with mental illness, especially teenagers our age. The website contains complexly placed and centred divs that work properly on both mobile and desktop, a top nav bar, a hamburger menu and multiple pages.
Creating this website was much more engaging than previous projects although - this being the first group coding work in the semester - we did not approach it optimally and had a perhaps uneven division of labour.
Link to the github repository .

#1: Snake Live Creating Coding Challenge

The Snake Creative Coding Challenge was by far the best and most interesting part of my experience:

  • It utilizes a language, JavaScript, which is fundamental in front-end development.
  • It was developed in a group and explored extremely complex computer science problems, headaches and bugs.
  • Unlike previous projects, the teamwork on this one was all but flawless.

During this project, we read through an implementation of the game Snake in P5.js and then worked to modify it as we saw fit; while starting with someone else's code may seem suboptimal, we ended up completely reengineering it regardless. At the end of the module, our group presented a "code walkthrough" of the original program and our modifications in what was, possibly, the first presentation in my life I ever enjoyed.
